# 지원금 매칭 PWA

대한민국 사업자를 위한 정책자금/지원사업 추천 웹앱입니다. 정적 파일만으로 동작하므로 GitHub Pages, Netlify, Vercel, Cloudflare Pages 같은 호스팅에 바로 올릴 수 있습니다.

## 실행

`index.html`을 브라우저에서 열면 바로 실행됩니다.

외부 공개 배포는 HTTPS 호스팅을 권장합니다. HTTPS에서 열면 PWA 설치와 서비스워커 캐시가 활성화됩니다.

## 현재 구현된 기능

- 첫 화면에서 예비창업자/기존 사업자 유형 선택
- 예비창업자 수기 조건 선택
- 기존 사업자 사업자등록증 이미지 OCR
- 기존 사업자 사업자등록증 PDF OCR
- OCR 추출값 기반 상호, 사업자번호, 개인/법인 유형, 대표자 생년월일 또는 법인등록번호, 지역, 국세청 업태/종목, 개업일 표시
- OCR 누락 시 개업일과 추천용 업종 분류 보정
- 샘플 지원사업 데이터 기반 매칭 점수 계산
- 적합도순, 마감 임박순, 지원 규모순 정렬
- 추천 사유, 확인 필요 조건, 필요서류 표시
- 관심사업 저장 버튼
- PWA 매니페스트, 앱 아이콘, 서비스워커 캐시

## 웹앱 배포

### GitHub Pages

1. 이 폴더의 파일을 GitHub 저장소에 올립니다.
2. 저장소 Settings > Pages에서 배포 브랜치를 선택합니다.
3. 배포된 HTTPS 주소로 접속합니다.

### Netlify 또는 Vercel

1. 새 프로젝트를 만들고 이 폴더를 업로드하거나 Git 저장소를 연결합니다.
2. Build command는 비워둡니다.
3. Publish directory는 프로젝트 루트로 설정합니다.

## Android 앱으로 만들기

가장 빠른 방식은 PWA를 Android에서 설치하는 것입니다. Chrome으로 HTTPS 배포 주소에 접속하면 홈 화면에 설치할 수 있습니다.

Play Store 배포가 필요하면 다음 순서가 좋습니다.

1. 먼저 웹앱을 HTTPS 도메인에 배포합니다.
2. Bubblewrap 또는 Capacitor로 Android 래퍼를 만듭니다.
3. 앱 이름, 아이콘, 패키지명을 설정합니다.
4. OCR, 개인정보 처리방침, 사업자정보 저장 여부를 정리한 뒤 Play Console에 제출합니다.
