## 현재 구현 상태

- `기업마당`: 실제 API 연동 완료
- `K-Startup`: 모집중 RSS 연동 완료
- `/api/programs`: 여러 데이터 소스를 합치기 위한 통합 API 추가
- 나머지 출처: API 키/공식 엔드포인트 확인 후 같은 어댑터 구조에 추가 예정

## 1차 연동 원칙

- 실제 API가 있는 곳부터 붙인다.
- API 키가 필요한 경우 프론트엔드에 직접 넣지 않는다.
- 공고 원문 링크를 항상 제공한다.
- 추천 결과에는 출처와 수집일을 표시한다.
- 조건이 불확실한 공고는 `추천 가능`이 아니라 `확인 필요`로 둔다.
