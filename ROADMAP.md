# 지원사업 매칭 앱 업그레이드 로드맵

이 앱의 목표는 예비창업자와 기존 사업자가 놓치기 쉬운 정부지원금, 정책자금, 보증, 고용지원, R&D, 창업지원 공고를 한곳에서 확인하고 본인 조건에 맞는지 빠르게 판단하게 돕는 것입니다.

## 현재 상태

- 예비창업자/기존 사업자 분기
- 사업자등록증 이미지/PDF OCR
- 개인사업자/법인사업자 구분
- 대표자 생년월일/법인등록번호 추출 시도
- 국세청 업태/종목 표시 및 보정
- 지역/업종/청년/여성기업/수출기업/창업 7년 이내 조건 기반 매칭
- 샘플 지원사업 데이터 기반 추천

현재 가장 큰 한계는 실제 공고 데이터가 아직 연결되지 않았다는 점입니다.

## 1단계: 실제 공고 데이터 구조 만들기

목표: 샘플 데이터를 실제 API/수집 데이터로 바꿀 수 있는 구조를 만든다.

- `supportPrograms` 하드코딩 제거 준비
- 공고 데이터 표준 스키마 정의
- 데이터 출처별 필드 차이를 하나의 형식으로 변환
- 공고마다 출처, 원문 링크, 수집일, 마감일, 지역, 업종, 대상, 지원유형 저장
- 앱 화면에 `출처`와 `업데이트 일자` 표시

우선 연동 후보:

- 기업마당
- K-Startup
- 소상공인24
- 중소벤처기업진흥공단
- 고용24

## 2단계: 기업마당 API 1차 연동

목표: 가장 범용적인 중소기업/소상공인 지원사업 공고를 실제로 가져온다.

- 기업마당 API 신청 및 인증키 확보
- 정적 앱에 API 키를 직접 넣지 않도록 서버/함수 구조 결정
- 공고 목록 수집
- 마감 지난 공고 제외
- 제목, 기관, 지역, 지원대상, 신청기간, 링크를 매칭 데이터로 변환
- 기존 샘플 데이터와 실제 데이터 병행 표시

권장 방식:

- GitHub Pages만으로는 API 키 보호가 어렵기 때문에 Netlify Functions, Vercel Functions, Cloudflare Workers 중 하나를 사용
- 프론트엔드는 공개 링크 유지
- API 키는 서버 환경변수로 숨김

## 3단계: 자격요건 판정 강화

목표: 단순 키워드 추천이 아니라 “왜 가능/불가능한지”를 설명한다.

- 지역 조건
- 업종 조건
- 업력 조건
- 대표자 연령 조건
- 개인/법인 조건
- 소상공인/중소기업 여부
- 매출/상시근로자 수
- 여성기업/장애인기업/사회적기업/벤처기업 인증 여부
- 기존 수혜 이력 여부

결과 상태:

- 추천 가능
- 확인 필요
- 대상 아님

대상 아님 공고는 기본 숨김 처리한다.

## 4단계: 예비창업자 전용 흐름 강화

목표: 아직 사업자등록 전인 사람에게도 의미 있는 추천을 제공한다.

- 창업 예정 지역
- 창업 예정 업종
- 대표자 연령
- 창업 예정 시점
- 예상 고용 규모
- 예상 매출 규모
- 창업 아이템 유형
- 오프라인 매장/온라인 사업/제조/R&D 여부

예비창업자 추천 우선순위:

- 창업교육
- 사업화 자금
- 예비창업패키지류
- 청년/여성/지역 창업지원
- 공간지원
- 멘토링/컨설팅

## 5단계: 기존 사업자 전용 흐름 강화

목표: 사업자등록증만으로 부족한 조건을 최소 질문으로 보완한다.

- OCR 추출값 확인
- 매출 규모
- 상시근로자 수
- 수출 여부
- 인증 여부
- 자금 필요 목적
- 희망 지원 유형

희망 지원 유형:

- 보조금
- 정책자금/융자
- 보증
- 고용지원금
- R&D
- 바우처
- 교육/컨설팅

## 6단계: 알림과 저장 기능

목표: 사용자가 마감일을 놓치지 않게 한다.

- 관심사업 저장
- 마감 임박 표시
- 신청 준비서류 체크리스트
- 캘린더 추가
- 이메일/카카오 알림은 후속 단계

## 7단계: 신뢰도와 법적 안내

목표: 금융/정책 추천 서비스로서 오해를 줄인다.

- 공고 원문 링크 제공
- 추천 사유와 제외 사유 명시
- “최종 신청 가능 여부는 기관 공고문 기준” 안내
- 개인정보/사업자정보 처리방침 준비
- OCR 파일 저장 여부 명시

## 8단계: 운영형 제품화

목표: 실제 사용자에게 지속 제공 가능한 서비스로 만든다.

- 백엔드 API
- 데이터베이스
- 공고 수집 배치
- 관리자 공고 검수 화면
- 사용자 계정
- 사업자 프로필 저장
- 정식 도메인
- 개인정보 처리방침
- 이용약관
